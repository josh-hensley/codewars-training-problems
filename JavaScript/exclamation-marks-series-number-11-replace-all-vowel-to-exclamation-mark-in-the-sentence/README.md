
        # Exclamation marks series #11: Replace all vowel to exclamation mark in the sentence
        
        ### Description:

 Replace all vowel to exclamation mark in the sentence. `aeiouAEIOU` is vowel.

### Examples

```javascript
"Hi!" --> "H!!"
"!Hi! Hi!" --> "!H!! H!!"
"aeiou" --> "!!!!!"
"ABCDE" --> "!BCD!"
```
